=========================
How to configure Asterisk
=========================

In ``/etc/asterisk/manager.conf``, add:

.. code-block:: ini

    [username]
    secret=password
    deny=0.0.0.0/0.0.0.0
    permit=127.0.0.1/255.255.255.255
    read = all
    write = all

Launch:

.. code-block:: sh

    $ rasterisk -x 'manager reload'
